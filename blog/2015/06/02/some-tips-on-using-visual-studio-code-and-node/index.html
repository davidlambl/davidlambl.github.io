Some Tips on Using Visual Studio Code and Node.js

Jun 2nd, 2015 9:41 pm | Comments

As a .NET Developer who uses a Mac at home, Visual Studio Code was pretty exciting news to me.

Moreover, Node.js has been hot for awhile now, and I’m still fairly ignorant to it,

The gist of the talk in the screenshot above is that JavaScript is rapidly careening toward world domination on its wobbly wheels.

And so, what a time to play with it!

There are just a few tips I’d like to point out here, should you decide to do the same,

  • There are annoying permissions errors I encounted after simply installing Node.js and following the Visual Studio Code walkthrough, when trying to save. I thought I had outsmarted this by remembering how to “sudo” commands, but that just exacerbated my problems by continually pushing them further down the line. This blog post helped me: www.johnpapa.net/how-to-use-npm-global-without-sudo-on-osx/
  • I first missed the bit about adding Code to your .bash_profile under the setup section. This is a nice thing to do; check out this blog post for help with that: natelandau.com/my-mac-osx-bash_profile/
